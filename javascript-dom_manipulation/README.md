# JavaScript - DOM Manipulation

Introduction à la manipulation du DOM avec JavaScript vanilla.

## Contenu

| Fichier | Description |
|---------|-------------|
| `0-script.js` | Change la couleur du header en rouge via le style inline |
| `1-script.js` | Change la couleur du header au clic sur un bouton |
| `2-script.js` | Ajoute la classe CSS `red` au header au clic |
| `3-script.js` | Bascule la classe du header entre `red` et `green` au clic |
| `4-script.js` | Ajoute un élément `<li>` à une liste au clic |
| `5-script.js` | Met à jour le texte du header au clic |
| `6-script.js` | Récupère le nom d'un personnage Star Wars via l'API SWAPI et l'affiche |
| `7-script.js` | Récupère la liste des films Star Wars via l'API SWAPI et les affiche |
| `8-script.js` | Affiche "Hello" dans la langue du navigateur en utilisant l'API HelloSalut |

## Concepts abordés

- Sélection d'éléments avec `querySelector`
- Modification de styles et de classes CSS
- Gestion d'événements avec `addEventListener`
- Création et ajout d'éléments dans le DOM
- Requêtes HTTP avec `fetch` et traitement des réponses JSON
