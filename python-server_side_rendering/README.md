# Python - Server Side Rendering

Introduction au rendu côté serveur en Python, avec génération dynamique de fichiers à partir de templates.

## Contenu

| Fichier | Description |
|---------|-------------|
| `task_00_intro.py` | Génère des fichiers d'invitation personnalisés à partir d'un template et d'une liste de participants |

## Concepts abordés

- Rendu côté serveur (Server Side Rendering)
- Manipulation de templates de texte
- Génération dynamique de fichiers
- Validation des entrées (type checking, gestion des cas vides)
